# preact-spotify-demo

Demo app desarrollada con [Preact](https://preactjs.com/) utilizando la [API de Spotify](https://developer.spotify.com/web-api/search-item).

Basada en el tutorial de [@impronunciable](https://github.com/impronunciable), se puede ver el tutorial [ac√°](https://www.youtube.com/watch?v=gXlJeMbH3dU).

## Usage

Install the dependencies
`$ npm i`

## Run the development server
`$ npm start`
